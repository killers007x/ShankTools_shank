single file:

python luaq_tool.py -d NameFile.lua


for rebuild:

python luaq_tool.py -c NameFile.lua -o NameFile.lua


 Decompile all files in folder:

python luaq_tool.py -db C:\Users\use_name\Desktop\luaq


 Compile all decompiled files:

python luaq_tool.py -cb C:\Users\use_name\Desktop\luaq
